# Pickiey.com

公開サイトは👉[コチラ](https://pickiey.com)

--------------------------------------------------------------------------------

## 使用ライブラリ

- フロントエンド
    - Vue  3
    - Vuex 4
    - Vue-router 4
- アニメーション
    - GSAP 3.6.0
- ビルドツール
    - Vite 2
- マークアップ
    - HTML ---> Pug
    - CSS  ---> Stylus

## セットアップと実行

```sh
git clone https://github.com/pickiey/vite.git 適当なフォルダ名
cd フォルダ名
npm install
npm run dev
```
